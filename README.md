# HW2Problem3_Paulino
A Java application that demonstrates the Chain of Resposibility design pattern with a JavaSwing UI element in a user authentication system scenario.

## Resources Used in Development
- Closer look at Email Verification (https://docs.oracle.com/javase/8/docs/api/java/lang/String.html)
- Simones code base from the presentation

## Design Patterns Implemented
### COR (Chain of Responsibility)
Used to go from verifier to verifier (password, username, email, phone) checking the information until everything is confirmed to be correct.

## Features
- Input Text Feilds for password, username, email, phone
- Protected password text input
- Optional phone number input
- Submission button
- verification checks

## Requirements
- JDK 8 or higher
- No external libraries

## Installation
1. Download src files and import into your desired directory.
2. Save and run the program to ensure installation was successful.

## Usage
Program will play automatically and prompt for user information. Follow the prompts to create a new user.
